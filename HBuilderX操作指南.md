# HBuilderX导入项目并生成APK详细步骤

## 第一步：导入项目

1. **打开HBuilderX**

2. **导入项目**
   - 点击顶部菜单栏：`文件` → `导入` → `从本地目录导入`
   - 或者使用快捷键 `Ctrl+O`

3. **选择项目目录**
   - 浏览并选择：`C:/Users/hopedloki/CodeBuddy/20251215113240`
   - 点击"选择文件夹"

4. **等待项目导入完成**
   - HBuilderX会自动识别这是一个UniApp项目
   - 项目会出现在左侧项目管理器中

## 第二步：检查项目配置

1. **查看manifest.json**
   - 在项目管理器中找到 `src/manifest.json` 文件
   - 确认appid已设置为：`__UNI__ARCAEA`
   - 应用名称：`Arcaea PTT计算器`

2. **检查依赖**
   - 打开 `package.json` 确认依赖已安装
   - 如果有红色波浪线，可以在HBuilderX终端中运行：`npm install`

## 第三步：预览项目（可选）

1. **运行到内置浏览器**
   - 右键项目 → `运行` → `运行到内置浏览器`
   - 检查项目是否正常运行

2. **运行到模拟器（可选）**
   - 右键项目 → `运行` → `运行到手机或模拟器` → `选择设备`

## 第四步：生成APK

1. **打开发行菜单**
   - 右键项目 → `发行` → `原生App-云打包`

2. **配置打包参数**
   - **平台**：选择 `Android`
   - **应用名称**：`Arcaea PTT计算器`
   - **包名**：`com.arcaea.ptt.calculator`（可以使用默认生成的）
   - **版本号**：`1.0.0`
   - **版本代码**：`100`

3. **选择证书**
   - 第一次使用可以选择 `使用DCloud公用证书`
   - 或者选择 `使用自有证书`（如果有）

4. **配置图标和启动图**
   - 可以选择默认图标，也可以自定义
   - 启动图可以暂时使用默认设置

5. **确认并开始打包**
   - 点击"打包"按钮
   - 等待云端处理（通常需要几分钟）

6. **下载APK**
   - 打包完成后会显示下载链接
   - 下载生成的APK文件

## 第五步：安装和测试

1. **传输APK到手机**
   - 通过数据线、邮件或其他方式将APK传输到Android设备

2. **安装应用**
   - 在手机上启用"未知来源应用安装"权限
   - 点击APK文件进行安装

3. **测试应用功能**
   - 打开应用测试各项功能
   - 检查登录、PTT计算、数据管理等功能

## 常见问题解决

### 1. 打包失败
- 检查网络连接是否正常
- 确认DCloud账号是否已登录
- 尝试重新配置打包参数

### 2. 依赖报错
- 在HBuilderX终端运行：`npm install`
- 或删除node_modules文件夹后重新安装

### 3. 编译错误
- 检查manifest.json配置是否正确
- 确认appid不为空

### 4. 云端打包限制
- 免费账号每日有打包次数限制
- 如需频繁打包可考虑升级为付费账号

## 替代方案：本地打包

如果云打包遇到问题，也可以尝试本地打包：

1. **配置Android SDK**
   - 确保已安装Android Studio
   - 设置ANDROID_HOME环境变量

2. **使用本地打包**
   - 右键项目 → `发行` → `原生App-本地打包`
   - 按照提示配置打包参数
